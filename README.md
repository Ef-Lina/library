# Библиотека
Это проект, который позволяет производить 
различные операции с данными о книгах. 
Сюда можно добавлять данные об исследовательских 
статьях, которые необходимо изучить, специальный 
алгоритм подберёт лучший список, что позволит 
изучить наибольшее количество статей, уделяя особое
внимание самым современным за ограниченный 
промежуток времени.

## Как использовать

1. Скачать этот репозиторий или клонировать его
2. Перейти в папку репозитория при помощи `cd`
3. `pip install -r requirements.txt`
4. `uvicorn src.main:app --reload`


## Версии ПО

Предполагается использование версии 0.115.12 
фреймворка FastAPI. Для ее запуска вам 
понадобится интерпретатор Python версии 
3.8, 3.9, 3.10, 3.11 или 3.12.
